# 💰 Expense Tracker Dashboard

A modern and responsive Expense Tracker Dashboard built using HTML, CSS, and JavaScript. This project helps users manage their daily income and expenses with an intuitive dashboard interface, real-time balance calculation, transaction history, charts, and local storage support.

---

## 🚀 Features

- Add Income and Expense Transactions
- Real-Time Balance Calculation
- Income and Expense Summary Cards
- Transaction History
- Search Transactions
- Filter by Income or Expense
- Category-Based Expense Tracking
- Monthly Statistics Chart
- Expense Category Pie Chart
- Dark Mode Toggle
- Responsive Dashboard Design
- Local Storage Support

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Local Storage API
